# Keyboard shortcuts

timedart is built to be driven entirely from the keyboard. Press `?` anywhere to open this list in-app.

## Navigation

Active in any list (side panel, tracker, settings):

| Keys | Action |
| --- | --- |
| `J` / `↓`, `K` / `↑` | Move down / up |
| `g` `g`, `G` | Jump to top / bottom |
| `L` / `→` | Open / expand |
| `H` / `←` | Collapse / parent |
| `Enter` | Select / activate |
| `e` | Edit the focused item |
| `Esc` | Back to list |

## Panes

| Keys | Action |
| --- | --- |
| `Tab` | Switch pane |
| `Ctrl`+`←` / `Ctrl`+`→` | Focus tracker / panel |
| `Ctrl`+`W` then `H` / `L` | Switch pane (vim-style) |
| `/` | Search |
| `Ctrl`+`,` | Open Settings |
| `?` | This help |

## Side panel

| Keys | Action |
| --- | --- |
| `A` | Add client |
| `a` | Add project |
| `n` / `N` | Next / previous project (search match) |

## Tracker

| Keys | Action |
| --- | --- |
| `t` | Open tracker |
| `Space` | Start / pause / resume the timer |
| `f` | Finish the current session |
| `a` | Add task |
| `A` | Add entry |
| `i` | Focus the description field |

## Editors

Inside an edit modal (client, project, task, or entry):

| Keys | Action |
| --- | --- |
| `Enter` / `Ctrl`+`S` | Save |
| `d` | Delete (only when no text field is focused) |
| `Esc` | Cancel / close |
